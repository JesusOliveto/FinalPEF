# Sistema de Recomendación de Rutas en una Ciudad Inteligente

Descripción:
Diseñar un sistema que sugiera la mejor ruta entre dos puntos en una ciudad, considerando tiempo, tráfico y distancia.

Requisitos técnicos:

Selección de algoritmo adecuado (Dijkstra, A*, BFS según el caso).

Uso de programación dinámica y memoización para recalcular rutas parcialmente.

Caching inteligente de rutas más usadas.

Uso de estructuras adecuadas (grafos con listas de adyacencia o mapas hash).

Concurrencia/paralelismo para procesar múltiples solicitudes de usuarios en paralelo.

Generadores (en Python) para iterar sobre caminos posibles de forma eficiente.

Herramientas de profiling para medir rendimiento y detectar cuellos de botella.

Implementación de batching para procesar varias consultas de rutas de forma eficiente.

Código con docstrings, comentarios y PEP 8 (en Python).

Aplicar refactorización para mejorar la legibilidad y modularidad.

Testing unitario y de integración.

IA: Implementar una heurística inteligente (aprendida de datos históricos de tráfico).
